# がい数ランド｜がい数の表し方と使い方

小学4年生「がい数の表し方と使い方」の単元を、ゲーミフィケーションつきで
段階的にマスターできる学習アプリ。

## モジュール（単元の知識・技能を段階的に網羅）

| モジュール | 内容 |
|---|---|
| がい数の いみ | 約何万・約何千を 数直線で読み取る／がい数を つかう場面の判断 |
| 四捨五入で がい数に | 指定の位まで・上から1けた／2けたの がい数、条件から数を選ぶ つまずきポイント |
| もとの数の はんい | いちばん小さい・大きい整数を求め、「以上・未満」で正しく表す |
| たし算・ひき算の 見積もり | 百の位までの がい数にしてから 計算する見積もり |
| かけ算・わり算の 見積もり | 上から1けたの がい数にしてから 計算する見積もり、代金の文章題 |
| 切り上げ・切り捨てで 考えよう | 場面に合う見積もり方の選択、予算内で買えるかの判定 |
| エラーハンター | 見る位・くり上がり・「上から◯けた」・はんい・見積もり方の混同、頻出誤答の発見・修正・理由の言語化 |
| 本番テストモード | 表（知識・技能100点）／裏（思考・判断・表現50点）を通しで挑戦 |

## ゲーミフィケーション

- きょうの目標リング（5/10/15/20問・タップで切替）
- 連続ノーミス記録・スキル別の熟達バー（5問連続ノーミスでMAX）
- ごほうびバッジ（累計・ノーミス・テスト得点・モジュール別デビュー／マスター・最終称号「【がい数】神」）
- バッジ獲得率で解放される特別テーマ（深海50%・極光75%・宇宙神100%）＋和・マトリックス
- おまかせモード（正答率にもとづく適応難易度。2問連続ノーミスでレベルアップ）
- ふくしゅうコーナー（正答率が低いスキルをホームで提案）
- やさしい効果音（Web Audio合成・設定でOFF可）とプロセス称賛メッセージ
- 学習のきろく（モジュール別クリア数・履歴・テストの答案詳細）

進捗・設定は localStorage に保存されます。

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
