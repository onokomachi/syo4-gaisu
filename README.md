# わり算ランド｜わり算の筆算ラボEX

小学4年生「わり算の筆算（÷1けた・÷2けた）」の単元を、ゲーミフィケーションつきで
段階的にマスターできる学習アプリ。

## モジュール（単元の知識・技能を段階的に網羅）

| モジュール | 内容 |
|---|---|
| あんざん わり算 | 九九の範囲 → あまりあり → 何十 → 何百（10・100のまとまり） |
| 筆算シミュレーター | たてる位置の自己選択・商の0・仮商の修正まで扱う中核モジュール |
| 商の見当づけ | 商がたつ位／商のけた数／四捨五入による仮商の見当 |
| たしかめ算 | わる数 × 商 ＋ あまり ＝ わられる数 の□うめ |
| わり算のきまり | 何十÷何十・何百÷何百・あまりの10倍注意・□うめ |
| ことばの もんだい | 等分除・包含除・あまりの切り上げ／切り捨て（式→計算→判断の段階入力） |
| エラーハンター | 商の0とばし・あまり≧わる数・ひき算ミス等の頻出誤答の発見・修正・理由の言語化 |
| 本番テストモード | 表（知識・技能100点）／裏（思考・判断・表現50点）を通しで挑戦 |

## ゲーミフィケーション

- きょうの目標リング（5/10/15/20問・タップで切替）
- 連続ノーミス記録・スキル別の熟達バー（5問連続ノーミスでMAX）
- ごほうびバッジ（累計・ノーミス・テスト得点・モジュール別デビュー／マスター・最終称号「【わり算】神」）
- バッジ獲得率で解放される特別テーマ（深海50%・極光75%・宇宙神100%）＋和・マトリックス
- おまかせモード（正答率にもとづく適応難易度。2問連続ノーミスでレベルアップ）
- ふくしゅうコーナー（正答率が低いスキルをホームで提案）
- やさしい効果音（Web Audio合成・設定でOFF可）とプロセス称賛メッセージ
- 学習のきろく（モジュール別クリア数・履歴・テストの答案詳細）

進捗・設定は localStorage に保存されます（旧バージョンの記録は初回起動時に自動引きつぎ）。

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
